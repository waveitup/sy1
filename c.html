C Language (UNIT - II)

History of C Language

- FORTRAN : Formula Translator.
- COBOL - Common Business Oriented Language.
- C Language was developed by Dennis Ritchie in early 1970s at AT&T Bell Laboratories.
- Initially it was designed for programming in operating system called UNIX. After the advent of C, the whole UNIX operating system was rewritten using it.
- C Language was derived from B Language which itself was derived from BCPL (Basic Combined Programming Language).
- BCPL was developed by Martin Richards at Cambridge University.
- In 1982, a committee was formed by ANSI (American National Standards Institute) to standardize C.
- Finally in 1989, standard for C was introduced (ANSI C).

Importance of C

i) C is commonly used programming language for system softwares and system level applications.
ii) Code written in C can be compiled and run on any type of computer with minimal changes.
iii) It's efficiency makes it suitable for programming micro controllers and other hardwares with limited resourses.
iv) C's ability to interact directly with hardware and perform bit-level operations makes it suitable for writing device drivers and firmwares.

Elements of C

1) Character Set
2) Keywords
3) Identifiers
4) Data Types
5) Variables
6) Constants

Character Set

Character Set is a set of characters that are used in C programs. It is a set of all the valid characters that we can use in the source program for forming words, expressions and numbers. C Language supports a total of 256 characters. Character set in C include alphabats, numbers, symbols etc.

Character Set include :
i) Alphanumeric Characters : Letters and Digits.
ii) Special Characters : Punctuations and Whitespaces.
iii) Escape Sequences : \t, \n, etc.
iv) Comments : Single line and Multi line.
v) Preprocessor Directives : #include, #define.

Keywords

Keywords are the reserved words in C for doing some specific task. They are the words with predefined meanings and can't be used as identifiers or variable names, as they have a specific function within the language. They are always written in lowercase. There are only 32 keywords in C language.

For Example : int, for, do, void etc.

Identifiers

Identifiers are the names assigned to the different entities such as variables and functions etc. Identifiers cannot be keywords. They are user defined words. They serve as symbolic references to memory locations or functions, facilitating code readability and maintainability.

Rules to name identifiers :
i) Name should consists of only letters, digits and underscore.
ii) First character should be an alphabat or underscore.
iii) Name should not be a keyword.
iv) C is a case sensitive. Uppercase and lowercase will be considered different.
v) An identifier's name may be arbitrarily long.

Data Types

Data Type is a classification that specifies which type of value a variable can hold and what kind of operations can be performed on that value. Data types play a crucial role in defining how data is stored, manipulated, and interpreted by the program.

Data Types in C :
i) char : Single character (1 byte)
ii) int : Interger numbers (2 bytes)
iii) float : Decimal numbers (4 bytes)
iv) double : Larger decimal numbers (8 bytes)
v) void : No value or return type.

Variables

Variable is a temporary container in which data can be stored within the computer memory. Basically, a variable in C is a symbolic name associated with a memory location that stores a value of a specific data type. This memory location allows the program to store, retrieve, and manipulate data during runtime. Variables are essential for performing operations, making decisions, and managing data flow within a program. They have a user defined name.

Declaration of a variable : int a;

Constants

Constants are the fixed values that do not change during the execution of the program. Once defined, their value remains the same. This prevents accidental changes and helps maintain the integrity of the data. For example, using a constant for mathematical values like Ï€ (pi).

Example : const int num = 10;

Symbolic Constant

Symbolic Constant in C language is a named constant whose value is fixed and does not change during the execution of a program. Symbolic constants are used to improve code readability and maintainability by giving meaningful names to constant values..

Example :
#define PI 3.14159 (using preprocessor directive)
const float PI = 3.14159; (using const)

Assignment Statement

Assignment Statement in C language is a fundamental operation used to set or update the value of a variable using assignment operator (=). It is essential for manipulating data and controlling the flow of a program.

Example : int num = 10;

scanf()

scanf is a commonly used input function in C Language. It allows you to read input from the user and store that input in a variable.

Example : scanf("%d", &a);

printf()

printf is a predefined function from the header stdio.h, it is used to print formatted output to the console.

Example : printf("Hello");

Operators

Operators are the symbols that perform operations on operands.

Types of Operator :

1) Arithmetic Operators : Used for methematical operations.
Ex : +, -, /, *, % etc.

2) Relational Operators : Used to compare two values.
Ex : <, >, ==, <=, >=, != etc.

3) Logical Operators : Used to perform logical operations.
Ex : Logical AND (a&&b)
Logical OR (a||b)
Logical NOT (!a)

4) Assignment Operators : Used to assign values to variables.
Ex : Simple assignment (a=b)
Addition assignment (a+=b is equivalent to a = a+b)
Subtraction assignment (a-=b is equivalent to a = a-b)

5) Increment and Decrement Operators : Used to increase or decrease the value of a variable by 1.
Ex : Increment (a++ or ++a)
Decrement (a-- or --a)

6) Bitwise Operators : Operate on binary representations of integers.
Ex : Bitwise AND (a&b)
Bitwise OR (a|b)
Bitwise XOR (a^b)
Bitwise NOT (~a)
Left shift (a<<1)
Right shift (a>>1)

7) Conditional (Ternary) Operators : A shorthand for the if-else statement.
Ex : (condition ? exp1 : exp2)

8) Comma Operator : Used to seperate expressions in places where multiple expressions are allowed.
Ex : int a = 5, b = 6, c += 1;

9) Pointer Operator : Used for pointer manipulation.
Ex : Dereference (*ptr)
Address of (&var)
